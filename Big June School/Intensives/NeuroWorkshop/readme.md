# NeuroWorkshop

В рамках июньской школы DevCon 2017 прошел 6-часовой интенсив по нейронным сетям и глубокому обучению. Все основные материалы мастер-класса
можно найти в отдельном репозитории http://github.com/shwars/NeuroWorkshop.
